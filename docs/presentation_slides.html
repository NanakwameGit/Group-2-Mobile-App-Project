April 2026
The Problem

Why do we need this app?

  • 💸
    Most people have no clear picture of where their money goes each month
  • 📱
    Existing finance apps are overcomplicated — accounts, budgets, charts, subscriptions
  • 🔒
    Many require cloud accounts and share your financial data with third parties
  • ✅
    We built something simple, private, and offline — all data stays on your phone
What It Does

Core Features

💳

Balance Dashboard

Gradient card showing current balance, total income, and total expenses — live updates.

📋

Transaction List

Scrollable list with category icons, dates, and colour-coded amounts (red/green).

➕

Add Transaction

Bottom sheet form — title, amount, category, date picker, income/expense toggle.

💾

Local Storage

Transactions survive restarts. JSON-encoded in SharedPreferences — no server needed.

What We Learned

Challenges & Solutions

  • ⏳
    App stuck on loading
    Missing WidgetsFlutterBinding.ensureInitialized() — platform channel not ready before SharedPreferences was called
  • 🎨
    Color type removed from Flutter 3.38 public API
    Replaced explicit Color annotations with inferred variables; pass isIncome: bool instead
  • 🔐
    GitHub push 403 — even after being added as collaborator
    Invitation was pending. Accepted via GitHub API: gh api --method PATCH user/repository_invitations/{id}
